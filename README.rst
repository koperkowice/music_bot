discord.py music bot
==========

1. install packs:


.. code:: sh

    # 1. discord.py
    py -3 -m pip install -U discord.py[voice]
    
    # 2. PyNaCl
    py -3 -m pip install -U PyNaCl

2. install source code:

- `source <https://github.com/koperkowice/music_bot/archive/refs/heads/main.zip>`_


3. edit a token in a token.json



4. run program:

.. code:: sh

    py -3 main.py

Commands
==========

*join - joins to voice channel

*disconnect - disconnects of channel

*play {url} - plays a music

*pause - pauses music

*resume - resumes music
